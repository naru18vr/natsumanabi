# なつまなび

中学2年生向けの夏休み宿題・英検4級・夏期講習・定期テスト管理PWAです。学習記録はブラウザの `localStorage` だけに保存し、外部へ送信しません。

## 起動

```bash
npm install
npm run dev
```

## 主な使い方

- 初回画面で学習時間と選択課題を設定
- 「今日」で完了、一部完了、明日に調整
- 「英検」で英検学習アプリのJSONまたは文章を取り込み
- 「報告」でGoogle Chat用の文章をコピー
- 「設定」で宿泊日・夏期講習を登録し、JSONバックアップを保存

サイトデータを消すと記録も消えるため、設定から定期的にバックアップしてください。

## GitHub Pages

リポジトリの Settings → Pages → Source を **GitHub Actions** にします。`main` へのpushでテスト・ビルド・公開を行います。HashRouterと相対baseを使うため、任意のリポジトリ名で動作します。
